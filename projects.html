Global Flame Ministry Web Platform

Type: Fullstack Web Application

Status: Live and in active development

A fully functional church management and content platform built for Global Flame Ministry, Jos, Nigeria. Simon designed and built the entire backend REST API using ASP.NET Core and PostgreSQL, and also developed the React/TypeScript frontend. The platform handles sermon management, blog publishing, event announcements, a giving and donation flow, and an admin dashboard. It is deployed on Heroku (backend) and Vercel (frontend) with Cloudflare DNS and a custom domain.

Tech stack

  • ASP.NET Core
  • PostgreSQL
  • React
  • TypeScript
  • Tailwind CSS
  • Heroku
  • Vercel
  • Cloudflare
  • Backblaze B2
