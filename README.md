# Simon Agboji — Student Portfolio

A multi-page personal portfolio website built as part of my CSC 106 (Introduction to Web Technologies) coursework at Miva Open University.

## Live Site

[View Portfolio](https://miva-agboji-simon.netlify.app)

## About the Project

I built this portfolio to document my academic background, showcase real-world projects I have worked on, and provide a functional academic task planner. The site is built with plain HTML, CSS, and JavaScript — no frameworks or libraries.

## Pages

- **Home** — Introduction, biography, academic profile, featured projects, and site navigation overview
- **About** — Educational background, career aspirations, technical skills, and personal interests
- **Projects** — Three projects with descriptions, tech stacks, screenshots, and links
- **Academic Planner** — Interactive task manager with add, complete, and delete functionality
- **Contact** — Contact form with client-side validation and direct contact links

## Tech Stack

- HTML5 (semantic elements throughout)
- CSS3 (Flexbox, CSS Grid, custom properties, animations)
- Vanilla JavaScript (DOM manipulation, form validation, event handling, arrays and functions)

## Projects Featured

- **Global Flame Ministry — Backend API** — REST API built with ASP.NET Core and PostgreSQL, deployed on Heroku. Handles sermon management, blog publishing, donation flows, and an admin dashboard.
- **Global Flame Ministry — Frontend** — React/TypeScript frontend deployed on Vercel. Live at globalflameministry.org
- **PIPDC Real Estate Platform** — Fullstack web application in progress for Plateau Investment and Property Development Company. Built with ASP.NET Core, React, TypeScript, and SQL Server.

## Project Structure

    student-portfolio/
      assets/       — CV and downloadable files
      css/          — External stylesheet (style.css)
      images/       — Project screenshots and profile photo
      js/           — JavaScript files (planner.js, contact.js)
      index.html    — Homepage
      about.html    — About Me page
      projects.html — Projects page
      planner.html  — Academic Planner page
      contact.html  — Contact page

## Running Locally

Clone the repository and open index.html in any browser. No build tools or dependencies required.

    git clone https://github.com/agbojisimon/student-portfolio.git
    cd student-portfolio

Then open index.html directly in your browser.
